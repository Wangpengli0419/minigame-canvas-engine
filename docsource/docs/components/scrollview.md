滚动内容的容器，滚动的前提是正确调用[updateViewPort](/api/api.html#updateviewport)
### 特殊样式
|      属性      |  类型  | 是否必填 |          说明          |
|----------------|--------|--------|------------------------|
| scrollX | string |   否  |  是否需要横向滚动，支持 "true"和"false"，默认值为 "false" |
| scrollY | string |   否  | 是否需要纵向滚动，支持 "true"和"false", 默认值为"false" |
