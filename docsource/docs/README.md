## 简介

原生js开发canvas界面困难而可维护性差，而引入一款游戏引擎又显得太过笨重。本项目旨在让开发者通过类Web开发的方式来降低canvas的开发门槛。

详细原理介绍文章：[https://segmentfault.com/a/1190000021297495?_ea=27021986](https://segmentfault.com/a/1190000021297495?_ea=27021986)
